# fase5-fundamentos-programacion
Solución informática en Python para la Fase 5 (Evaluación Final POA) del curso Fundamentos de Programación - UNAD. El programa almacena datos de sesiones de clientes en una matriz y evalúa automáticamente su nivel de compromiso (engagement) mediante el uso de estructuras repetitivas, arreglos bidimensionales y funciones modulares.
